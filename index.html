Booking Calendar JS

A lightweight JavaScript booking calendar with time slot selection. Perfect for appointment booking systems like salons, clinics, and services.

Live Demo Documentation

Features

📅 Interactive Calendar

Select dates easily with a modern UI calendar interface.

⏰ Time Slot Booking

Generate available booking slots automatically.

🌙 Dark & Light Theme

Built-in theme switching with modern styling.

🎨 Custom Primary Color

Match the calendar with your brand color.

🚫 Disable Past Dates

Prevent users from booking past dates.

⚡ Lightweight

No dependencies. Pure vanilla JavaScript.

🔁 Multiple Instances

Use multiple calendars on the same page.

📱 Responsive

Works perfectly on desktop and mobile devices.
