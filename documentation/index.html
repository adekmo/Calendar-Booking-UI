Booking Calendar Plugin

Lightweight JavaScript booking calendar with time slots.

Installation

Include the CSS and JavaScript files inside your project.

<link rel="stylesheet" href="../dist/booking-calendar.css">
<script src="../dist/booking-calendar.js"></script>

Add a container where the calendar will be rendered.

<div id="calendar"></div>

Basic Usage

const calendar = new BookingCalendar("#calendar", {

theme: "light",

primaryColor: "#6366f1",

startHour: 9,
endHour: 17,

bookedSlots: {
"2026-03-12": ["09:00","10:30"]
}

});

Configuration Options

  • theme — Calendar theme ("light" or "dark")
  • primaryColor — Main accent color
  • startHour — Start booking hour
  • endHour — End booking hour
  • slotDuration — Time slot interval (minutes)
  • bookedSlots — Predefined booked slots
new BookingCalendar("#calendar",{
theme:"dark",
primaryColor:"#22c55e"
})

Methods

  • getSelectedDate() → return selected date
  • getSelectedSlot() → return selected time
  • addBooking(date,time) → add booked slot
  • clearSelection() → clear selection
  • destroy() → destroy calendar instance
calendar.addBooking("2026-03-20","10:00");

Events (Callbacks)

You can listen to calendar events.

new BookingCalendar("#calendar",{

onDateSelect:(data)=>{
console.log("Date:",data.date)
},

onSlotSelect:(data)=>{
console.log("Slot:",data.time)
}

})

Examples

Dark Mode

new BookingCalendar("#calendar",{
theme:"dark"
})

Custom Color

new BookingCalendar("#calendar",{
primaryColor:"#22c55e"
})

Multiple Calendars

new BookingCalendar("#calendar1");
new BookingCalendar("#calendar2");
